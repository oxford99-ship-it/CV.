# Hak Seung Lee CV

This repository contains a simple HTML/CSS CV website.

## Files

- `index.html`: main CV webpage
- `style.css`: webpage styling

## GitHub Pages

Upload `index.html` and `style.css` to the repository root, then enable GitHub Pages from:

Settings → Pages → Deploy from a branch → main → /root → Save

## Save as PDF

Open the deployed CV website and press `Ctrl + P`.

Recommended settings:

- Destination: Save as PDF
- Paper size: A4
- Margins: Default
- Scale: 85–95%
- Background graphics: On
