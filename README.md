## Selected Projects

### CEVSim: Cosmetic Emulsification Vessel Scale-Up Simulator  
**Smart Factory Capstone Design Project** · Mar 2026 – Jun 2026

- Developed a web-based digital twin simulator for cosmetic emulsification processes using **Three.js** and **Population Balance Modeling**.
- Implemented a 65-node Fixed-Pivot PBM engine to predict droplet size distribution, viscosity, hardness, and emulsion uniformity.
- Built a real-time 3D simulation environment reproducing homogenizer, paddle, and scraper impeller dynamics.
- Designed a multi-metric quality evaluation framework integrating **D32, H-Index, viscosity, and hardness**.
- Developed scale-up prediction logic for **100 L, 300 L, and 500 L** manufacturing vessels based on constant energy dissipation rate principles.
- Validated simulation outputs against real manufacturing batches.

#### Research Outputs

- Submitted an industry track paper to ICSOC 2026 titled **“Scale-Up Simulation of Cosmetic Emulsion Manufacturing Vessels (100–500 L) Using a Population Balance Model: Integrated Multi-Metric Analysis of Droplet Size Distribution, Uniformity, Viscosity, and Hardness.”**
- Filed a patent application titled **“Method and System for Predicting and Evaluating Scale-Up Emulsification Quality of Cosmetic Emulsion Manufacturing Vessels Using Population Balance Models and Multi-Quality Indicators.”**  
  **Reference No. R-2026-0578-KR-1**, filed Jun. 2026.
- Registered the industrial software **CEVSim: Cosmetic Emulsification Vessel Scale-Up Simulator** for software copyright protection.
