CEVSim: Cosmetic Emulsification Vessel Scale-Up Simulator

Mar 2026 – Jun 2026

Smart Factory Capstone Design Project

Project Summary

  • Developed a web-based digital twin simulator for cosmetic emulsification processes using Three.js and Population Balance Modeling.
  • Implemented a 65-node Fixed-Pivot PBM engine to predict droplet size distribution, viscosity, hardness, and emulsion uniformity throughout the emulsification process.
  • Built a real-time 3D simulation environment reproducing homogenizer, paddle, and scraper impeller dynamics with particle-based flow visualization.
  • Designed a multi-metric quality evaluation framework integrating D32, H-Index, viscosity, and hardness, enabling automated PASS / REVIEW / FAIL classification.
  • Developed scale-up prediction logic for 100 L, 300 L, and 500 L manufacturing vessels based on constant energy dissipation rate principles.
  • Validated simulation outputs against real manufacturing batches and demonstrated applicability to smart manufacturing and process optimization.
